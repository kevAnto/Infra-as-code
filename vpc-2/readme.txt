We can setup AWS VPC with the single open-source terraform module provided by the community.
With the terraform-aws-modules terraform module we can create the following resources: VPC, Subnet, Route, Route table, Internet Gateway, Network ACL, NAT Gateway, VPN Gateway, VPC Flow Log, VPC Endpoint. 
It accepts 404 input variables, that allow you to customize your environment to suit your needs. 
It saves you time and enforce you to follow best practices from AWS.
